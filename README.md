# フリマ閲覧済みチェッカー（みちゃった君）

フリマサイトで以前閲覧した商品を自動で検出し、検索一覧や商品ページでマークを表示するChrome拡張機能です。

**対応サイト:**
- メルカリ（通常商品・メルカリショップ）
- 楽天ラクマ
- 楽天市場
- ヤフオク
- PayPayフリマ

## 機能

- **自動記録**: 商品ページを開くと自動的に閲覧履歴を保存
- **一覧でマーク表示**: 検索結果で閲覧済み商品にバッジを表示
- **商品ページでも通知**: 再訪問時にタイトル上部にバッジを表示
- **クロスサイト連携**: PayPayフリマで見た商品がヤフオク検索でも閲覧済み表示
- **手動登録**: ポップアップから商品IDやURLを直接登録可能
- **無制限保存**: 登録件数の上限なし
- **全削除機能**: ワンクリックで全履歴を削除可能

### 手動インストール
1. このリポジトリをダウンロードまたはクローン
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

### 自動記録
対応サイトの商品ページを開くだけで自動的に記録されます。

### 手動登録
1. ツールバーの拡張機能アイコンをクリック
2. 商品URLまたはIDを入力（複数行で一括登録可）
3. 「登録する」をクリック

### 対応形式
- メルカリ: `https://jp.mercari.com/item/m12345678901`
- メルカリショップ: `https://jp.mercari.com/shops/product/xxxxx`
- ヤフオク: `https://page.auctions.yahoo.co.jp/jp/auction/xxxxx`
- PayPayフリマ: `https://paypayfleamarket.yahoo.co.jp/item/xxxxx`

## プライバシー

- すべてのデータはブラウザ内に保存されます
- 外部サーバーへのデータ送信は一切行いません
- 閲覧履歴はお使いのブラウザ内にのみ保存されます
